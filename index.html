Calculator

    Calculator (계산기)

  • 사칙연산 계산기를 만들다

    • 사칙연산 계산기 사용방법

  • 버튼을 이용하여 데이터를 입력
  • "=" 버튼을 눌러 결과 값 출력
  • "del" 버튼으로 최근 입력값 지우기
  • "AC"버튼으로 전부 지우기

    • 추가 기능

  • 괄호 사용으로 조금 더 복잡한 사칙연산 가능
  • 계산 결과에서 데이터 입력하여 계속 연산 가능

    • ex) 3 + 2 = 5 추가로 +2 를 하면 5+2 = 7이 반환

  • 계산 기록 저장

    • 계산 기록칸이기 때문에 계산 결과 출력시 텍스트는 사라집니다
